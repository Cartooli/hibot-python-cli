## 🎯 Common Workflows

### "I want to learn HTML/CSS"
1. Load a **Template** (Quick Start section)
2. Accept a **Challenge** (Learn by Doing)
3. Toggle **Learning Hints** ON
4. Experiment with **Layout Helpers**
5. Watch your **Achievement Badges** unlock!

### "I want to deploy to Carrd"
1. Create your page
2. Click **Carrd** button (in Export for Platforms)
3. Read the modal instructions
4. Code is already copied to clipboard!
5. Go to Carrd and paste

### "I want to keep working elsewhere"
1. Click **Replit**, **CodeSandbox**, or **Glitch**
2. Code is copied automatically
3. Platform opens in new tab
4. Paste and continue coding!

### "I want to share my work"
1. Click **GitHub Gist** button
2. GitHub opens with code copied
3. Create your gist
4. Share the URL!

## 🎉 Getting Started

**First-time user? Try this:**

1. Open the app
2. Select **"Challenge 1: Change 3 Colors"**
3. Follow the hint that appears
4. Use the color pickers in the panel
5. Watch the "Color Master" achievement unlock!
6. Click **"CSS Grid"** in Layout Helpers
7. See instant layout code added
8. Click **"Carrd"** to see export instructions
9. Celebrate your progress! 🎊

**That's it! You've explored the core new features.**
