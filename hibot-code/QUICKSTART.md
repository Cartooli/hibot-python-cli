# Quick Start Guide

## 🚀 Get Started in 30 Seconds

### Option 1: Open Directly (No Server Needed!)
Simply double-click `index.html` and open it in your browser. That's it!

### Option 2: Run with Local Server
```bash
./start.sh
```
Then open: http://localhost:8000

---

## ✨ What You'll See

The **Hi Bot Code** interface has three main areas:

1. **Live Preview** (top-left): Your webpage renders here in real-time
2. **Edit Panel** (top-right): Visual controls to customize your page
3. **Code Editor** (bottom): The actual HTML code that powers everything

---

## 🎯 Try These First Steps

1. **Change the headline:**
   - Find "H1 (main headline)" in the Edit Panel
   - Type something new like "My Awesome Website!"
   - Watch it update instantly in the preview

2. **Pick a color:**
   - Click the "Accent" color picker
   - Choose your favorite color
   - See the button and headings change color

3. **View the code:**
   - Look at the "HTML Code" editor at the bottom
   - See how your changes updated the actual HTML
   - Try typing directly in the code editor!

4. **Pop out the preview:**
   - Click the "Pop-out" button
   - Your page opens in a new window
   - Save it or share it with others

---

## 📚 Learning Path

### Beginner (5 minutes)
- Change all the text fields
- Experiment with colors
- Try different font families

### Intermediate (15 minutes)
- Click "Update Code from Panel" to see how the HTML changes
- Edit the code directly in the bottom editor
- Click "Sync Panel from Code" to update the controls

### Advanced (30+ minutes)
- Add new HTML elements in the code editor
- Create custom CSS styles
- Add JavaScript for interactivity

## 💡 Tips & Tricks

- **Auto-save**: Changes in the panel automatically update the code

- **Offline capable**: No internet needed after initial load
- **Mobile friendly**: Works on phones and tablets too!

## ❓ Common Questions

**Q: Do I need to install anything?**
A: Nope! Just a modern web browser.

**Q: Can I save my work?**
A: Click "Pop-out" to open your page, then File → Save (Ctrl+S) to download the HTML.

**Q: Will my data be sent anywhere?**
A: No! Everything runs 100% locally in your browser.

**Q: Can I use this for real websites?**
A: Absolutely! The generated HTML works anywhere.

**Q: How do I deploy to my domain?**
A: See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed instructions.

## 🆘 Troubleshooting

**Preview not updating?**
- Click the "Run Code" button
- Make sure you clicked "Update Code from Panel"

**Colors not working?**
- Some color combinations need both background and text adjusted
- Try the preset font families first

**Code looks broken?**
- Click "Reset Example" to start fresh
- Check the browser console for errors (F12)
