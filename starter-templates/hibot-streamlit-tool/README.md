# hibot-streamlit-tool

> Pure Python UI without HTML/CSS. Streamlit turns Python scripts into interactive web apps — build study tools and calculators fast.

Built with **Streamlit (Python)**.

## Quick Start

1. Clone: `git clone https://github.com/Cartooli/hibot-streamlit-tool.git`
2. Install: `pip install streamlit`
3. Run: `streamlit run app.py`
4. Customize: Edit the Python code to add your own logic
5. Deploy: Use Streamlit Cloud or Vercel

- Core concepts of Streamlit (Python)
- How hibot-streamlit-tool is structured
- How to customize and deploy your own version
