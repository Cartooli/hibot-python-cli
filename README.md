# hibot-python-cli

> A command-line tool with zero dependencies. Learn file I/O, loops, and data structures by building a terminal game.

Built with **Python (no dependencies)**.

## Quick Start

1. Clone: `git clone https://github.com/Cartooli/hibot-python-cli.git`
2. Run: `python main.py`
3. No install needed — just Python
4. Customize: Add your own questions, scoring logic, or features

## What You'll Learn

- Core concepts of Python (no dependencies)
- How hibot-python-cli is structured
- How to customize and deploy your own version

## See It Live

Open the demo and see it in action, then rebuild it your way on [code.hibot.space](https://code.hibot.space/edtech-tutorials.html).
